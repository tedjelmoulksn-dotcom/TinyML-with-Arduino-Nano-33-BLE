# Part 2 – Electronic Component Classification with Edge Impulse

## Overview

This part of the project demonstrates how to build a **TinyML computer vision system** capable of recognizing electronic components using an **Arduino Nano 33 BLE and an OV7670 camera**.

A machine learning model is trained using **Edge Impulse** and then deployed on the Arduino for **real-time inference**.

The system detects components such as:

- LEDs
- Resistors
- Capacitors

The detected components are then transmitted to **Node-RED**, where they are counted and displayed on a dashboard.
